# Skeleton

**Skeleton** is an opinionated starter kit for building modern web applications with
Laravel, Inertia.js, and Vue.js. It provides a solid foundation with pre-configured
tools and best practices to help you get started quickly and efficiently.

## Usage

To create a new Laravel project using the Skeleton starter kit, run the following command:

```bash
laravel new --using nhedger/skeleton your-project-name
```
